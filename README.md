\# Schulbibliothek – Ausleihsystem



\*\*Autor:\*\* Ahmet Aktas



\## Kurzbeschreibung

Ein GUI-Programm zur Verwaltung der Buchausleihe in einer Schulbibliothek.

Schüler und Lehrer können Bücher suchen, ausleihen und zurückgeben.

Bibliothekare verwalten den Bestand und erstellen Mahnungen für überfällige Bücher.
